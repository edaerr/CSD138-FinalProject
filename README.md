## ‚ñ∂Ô∏è How to Run the SQL Script

1. Open **MySQL Workbench**  
2. Create a new SQL tab  
3. Open `FinalProject.sql`  
4. Select *the entire script* (Ctrl+A / Cmd+A)  
5. Run it (lightning bolt icon)  
6. The script will:  
   - Drop the database (if it exists)  
   - Recreate the full schema  
   - Insert all sample data  
   - Create indexes  
   - Create the subquery, view, stored procedure, and stored function  
   - Run verification queries  

If the script runs without errors, the database is correctly implemented.
